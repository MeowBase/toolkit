Converts the long size to human-readable string.

For precision 1 example:

                                 SI     BINARY

                      0:        0 B        0 B
                     27:       27 B       27 B
                    999:      999 B      999 B
                   1000:     1.0 kB     1000 B
                   1023:     1.0 kB     1023 B
                   1024:     1.0 kB    1.0 KiB

                7077888:     7.1 MB    6.8 MiB
              452984832:   453.0 MB  432.0 MiB
