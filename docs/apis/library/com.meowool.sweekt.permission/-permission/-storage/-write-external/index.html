WriteExternal

object WriteExternal : Permission.Single
Content copied to clipboard

Allows an application to write to external storage.

Note: If both your minSdkVersion and targetSdkVersion values are set to 3 or lower, the system implicitly grants your app this permission. If you don't need this permission, be sure your targetSdkVersion is 4 or higher.

Starting in API level 19, this permission is not required to read/write files in your application-specific directories returned by Context.getExternalFilesDir and Context.getExternalCacheDir.

If this permission is not allowlisted for an app that targets an API level before Build.VERSION_CODES.Q this permission cannot be granted to apps.

For AndroidManifest.xml: android.permission.WRITE_EXTERNAL_STORAGE.
